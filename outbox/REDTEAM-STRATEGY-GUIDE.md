## 1. DocsHunt.ai 샘플 분석 결과

- **샘플 URL:** https://docshunt.ai/sample
- **페이지 수:** 25페이지
- **품질:** AI 생성 수준 높음 (RAG + Fine-tuning 적용)

### 1.2 문제점 (심사위원 관점)
| 항목 | 문제 | 심사위원 반응 |
|------|------|--------------|
| **표현** | "혁신적", "혁명적" 과다 사용 | 허세로 감점 |

| **톤** | 완벽한 문장 구조 | AI 생성 티가 남 |
| **고객 인용** | 과장된 피드백 | 진위성 의심 |

### 1.3 시사점
> **"AI가 쓴 계획서"는 티가 난다.**  
> 20·30대 창업희망자라면 누구나 AI로 계획서 써볼 것.  
> 하지만 심사위원은 "또 AI 계획서" 하고 감점할 것.

## 2. Dual-Track 비즈니스 모델

### 2.1 트랙 구분

| 구분 | Track 1: 청년창업 | Track 2: 중장년 소기업 |
|------|------------------|---------------------|
| **타겟** | 20·30대 | 40·50대 |
| **사업 유형** | SaaS/플랫폼 | 기술개발/제조 |
| **예산** | ₩100만~500만 | **천만원 기준** |
| **정보 접근** | 높음 (AI 능숙) | **낮음 (정보격차 큼)** |
| **보유 자원** | 아이디어 | **기술/노하우/거래처** |
| **사업 목표** | 성장/스케일 | **원가절감/불량률 하락/거래선 확장** |
| **채널** | 온라인 (SNS, 유튜브) | 오프라인 (협회, 박람회) |
| **클로징** | 온라인 자동결제 | 오프라인 미팅/계약 |

### 2.2 우리의 포지션: 중간의 빈자리

**"온라인으로 발견하고, 오프라인 신뢰로 클로징"**

| 항목 | SaaS | 컨설팅 | **우리 (중간)** |
|------|------|--------|----------------|
| **가격** | ₩3만~10만/월 | ₩1,000만~5,000만 | **₩100만~300만** |
| **접근** | 셀프서비스 | 전담 매니저 | **하이브리드** |
| **품질** | AI 생성 | 전문가 작성 | **AI + 전문가 검수** |
| **시간** | 즉시 | 2~4주 | **3~7일** |
| **마감** | 온라인 | 오프라인 | **온라인 → 오프라인** |

### 2.3 차별화 포인트
- **DocsHunt:** 청년창업자만 (AI 생성 중심)
- **우리:** 청년 + 중장년 모두 (AI + 전문가 검수)
- **중장년 세그먼트:** 예산 있음, 정리는 못함, 실용적 목표 선호

## 3. REDTEAM 운영 체계

### 3.1 역할 분담

| 단계 | OpenClaw (Local) | Kimi Claw (Hosted) |
|------|-----------------|-------------------|
| **초안 생성** | 기술 구현, 데이터 처리 | 콘텐츠 작성, 전략 수립 |
| **검증 (REDTEAM)** | AI 흔적 탐지, 심사 기준 검토 | 시장 적합성, 경쟁사 비교 |
| **피드백** | 기술적 개선점 | 전략적 수정안 |
| **최종 검수** | 자동화 품질 체크 | 인간적 완성도 검토 |

### 3.2 번갈아 가는 검증 프로세스

```
[OpenClaw] 초안 생성
    ↓
[Kimi Claw] 1차 검토 (시장/전략 관점)
    ↓
[OpenClaw] 2차 검토 (기술/품질 관점)
    ↓
[Kimi Claw] 최종 검수 (인간적 톤 체크)
    ↓
[사용자] 승인/수정 요청
```

### 3.3 검증 체크리스트

**AI 흔적 탐지:**
- [ ] "혁신적", "혁명적", "차세대" 등 과장 표현 없음
- [ ] 현실적인 목표 수치 (근거 제시)
- [ ] 구체적인 실행 계획 (월별/분기별)
- [ ] 자연스러운 문장 구조 (AI 톤 탈출)

**시장 적합성:**
- [ ] 타겟 고객 명확화
- [ ] 경쟁사 차별화 포인트
- [ ] 수익 모델 구체성
- [ ] 위험 요소 및 대응 전략

**심사 기준 충족:**
- [ ] 공고 양식 준수
- [ ] 필수 항목 누락 없음
- [ ] 증빙 자료 연계성
- [ ] 가독성 및 구조화
