## 📋 조사 개요

**조사 목적:**

1. 비즈니스온 (BusinessOn) 및 유사 SaaS

**조사 방법:**

- 공식 웹사이트 분석
- 가격 및 기능 비교

## 📝 상세 분석

### 1. ZUZU (주주네트워크)

**기본 정보:**
- **타겟:** 예비창업자, 창업 3년 이내 기업
- **서비스 유형:** AI 피드백 중심
- **가격:** 미공개 (컨설팅 연계 추정)

**강점:**
- 심사위원 관점의 피드백 제공
- 객관적 평가 시스템
- 전문가 네트워크 활용

**약점:**
- 자동 생성 기능 부재
- 초기 초안 작성 어려움
- 높은 가격대 예상

**우리와의 비교:**
- ZUZU: 피드백 중심 → **우리: 생성 + 피드백 통합**

## 🎯 요약 5줄

1. **시장은 분화 중:** 피드백 중심(ZUZU), 공고 수집 중심(Quantum), 데이터 분석 중심(Enhans)으로 특화된 서비스들이 각자 영역을 구축 중

2. **가격대는 ₩5만~10만/월:** 대부분 구독형이며, 정부지원사업 예산 범위 내에서 책정됨

3. **공공 서비스와의 차별화 필요:** 정부24/K-Startup은 물으지만 AI 기능이 없어 이를 보완하는 것이 관건

4. **산업별 특화가 트렌드:** 범용 AI보다 제조, 법무, 의료 등 특정 산업의 전문 용어와 서식을 이해하는 AI가 경쟁력 있음

5. **단순 생성에서 피드백/컨설팅으로:** AI가 쓰는 것을 넘어, 심사위원 관점에서 평가하고 개선 방향을 제시하는 것이 차별화 포인트

## 💡 우리 서비스와의 차별화 포인트 분석

| 경쟁사 강점 | 우리 차별화 방향 |
|-------------|------------------|
| ZUZU: 피드백 중심 | → **자동 생성 + 피드백 통합**: 쓰고, 검토하고, 수정하는 일체형 워크플로우 |
| Quantum: 공고 수집 | → **사업계획서 ↔ 공고 자동 매칭**: 공고별 맞춤 사업계획서 섹션 자동 조정 |
| Enhans: 산업별 특화 | → **업종별 템플릿 + 예시 DB**: 50개 이상 업종별 실제 선정 사례 학습 |
| 정부24: 공식 데이터 | → **실시간 연동 + AI 해석**: 공고 변경사항 알림, 해석 제공 |
| 개인 AI: 저렴함 | → **합리적 가격 + 전문성**: ₩3만~5만/월대에 컨설팅급 품질 |

### 핵심 차별화 전략

**"AI 작성 + REDTEAM 검증 + 실시간 피드백"**

1. **Multi-Agent 구조**: 작성 에이전트 + 검증 에이전트 + 피드백 에이전트가 협업
2. **선정 사례 기반**: 실제 선정된 사업계획서를 학습시킨 파인튜닝 모델
3. **동적 템플릿**: 공고별, 업종별, 지역별로 최적화된 템플릿 자동 생성
4. **평가 시뮬레이션**: 심사위원 관점에서 자체 평가 및 개선점 제시
